Basics

The layout works with support of flexbox. With multi-column flex layout there is known bug, that if the number of items is not even in each row, the last row's items that is last will stretch across the whole container - this is not the case and is fixed.

Changing color of buttons on hover is done via css filter property.

Don't expect it to work in EDGE 12, but later ones will do. Probably does not work for Opera (whatever it is) either.

Size around 4kb, very efficient.

Buttons

Use the base class .btn.

Regular buttons

Use the following code: 

<button class="btn btn-primary">Primary</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-success">Success</button>
<button class="btn btn-warning">Warning</button>
<button class="btn btn-danger">Danger</button>

Disabled buttons

Add [disabled] property, .disabled classname or [aria-disabled="true] attribute.

Outlined buttons

Just add .outline classname.

Sizing

Use .xs, .sm, .lg or .gi
